measure(string)

Executes the next step in the run’s state machine:

  • FIRST_AIR: records min/max air and verifies
  • FIRST_SAMPLE: auto-gains, measures sample, verifies, stores
  • AIR: measures air and verifies
  • SAMPLE: measures sample, verifies, stores, and returns to AIR
Saves the updated JSON after each step.
